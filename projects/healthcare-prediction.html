Healthcare Risk Prediction

Predicting heart disease using machine learning models

📌 Project Overview

Built predictive models for identifying patients at high risk of heart disease using clinical data and patient records.

🧠 Techniques Used

  • SVM, Naive Bayes, Logistic Regression
  • K-Means for patient segmentation
  • Data visualization with seaborn & matplotlib
  • Accuracy: ~85%

📁 Key Features

  • Exploratory data analysis
  • Clustered risk stratification
  • Clean ML pipeline
