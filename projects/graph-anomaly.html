Graph-Based AML Anomaly Detection

Detecting suspicious financial transactions using graph analysis

📌 Project Overview

This project builds a graph of transactions and applies node embeddings + clustering to detect anomalous financial behavior for AML purposes.

🧠 Techniques Used

  • Graph construction from transaction data
  • Node2Vec embeddings
  • K-Means clustering
  • Isolation Forest for anomaly scoring

📈 Results

Several nodes with high anomaly scores were flagged. Graph-based features improved risk targeting vs. flat transaction features.
