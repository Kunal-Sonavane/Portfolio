Credit Risk Modeling

Explaining loan default risk using ML + SHAP

📌 Project Overview

This project builds an explainable machine learning model to predict loan defaults using structured credit data.

🧠 Techniques Used

  • XGBoost Classifier
  • Hyperparameter tuning with GridSearchCV
  • SHAP for model explainability
  • Confusion Matrix, ROC-AUC

📈 Results

The XGBoost model achieved an ROC-AUC score of 0.74. SHAP values helped identify key drivers like loan amount, income, and credit history.

📁 Key Features

  • Realistic credit risk dataset
  • Full preprocessing pipeline
  • SHAP summary & waterfall plots
