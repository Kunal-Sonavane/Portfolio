Market Basket Analysis

Finding product combinations using Apriori algorithm

📌 Project Overview

This project identifies product relationships in 38,000+ retail transactions using association rule mining.

🧠 Techniques Used

  • Apriori algorithm
  • Frequent itemsets with minimum support
  • Lift, confidence, and rule pruning
  • Bar charts and heatmaps for support & lift

📈 Results

Generated strong cross-selling rules like "Bread → Butter" and "Milk → Tea" with confidence > 70% and high lift.
