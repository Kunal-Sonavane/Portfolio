FX Forecasting (INR/USD)

Using ML & Macroeconomic Indicators to Predict Currency Movement

📌 Project Overview

This project forecasts INR/USD exchange rates using both traditional econometric models and machine learning algorithms.

🧠 Techniques Used

  • Random Forest, XGBoost Regressor
  • ARIMA and VAR models
  • Cointegration Tests, ADF, Granger Causality
  • Feature importance with SHAP

📈 Results

The hybrid models combining macroeconomic indicators with ML achieved robust predictive performance with improved generalization.

📁 Key Features

  • RBI macroeconomic indicators
  • ML + Econometrics combo
  • Visual trend comparison
